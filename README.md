# fictional-character-ontology
Repository for the data &amp; code of the paper [Toward an Ontological Representation of Fictional Characters}(https://www.cambridge.org/core/services/aop-cambridge-core/content/view/EF9F289D4EA7FC0A6ECC38BAA391DF76/S2977815826100256a.pdf/toward-an-ontological-representation-of-fictional-characters.pdf)

Characters are central to narrative theory but remain under-specified in computational work, where they are often reduced to clusters of words or vectors. We propose an operationalizable ontology of characterization that bridges narratological theory and NLP.
